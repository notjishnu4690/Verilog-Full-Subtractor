# Full Subtractor using Verilog HDL

## Overview
This project implements a Full Subtractor using Verilog HDL and verifies its functionality through simulation.

## Inputs
- A
- B
- Bin (Borrow In)

## Outputs
- Difference
- Borrow

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave

## Files
- fullsubtractor.v : Full Subtractor design
- tb5.v : Testbench
- fullsubtractorwaveform.jpg : Simulation waveform

## Simulation Result
The design was successfully simulated and verified using the provided testbench and waveform output.
